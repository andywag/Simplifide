This plugin supports an autoconnect mode similar to the Verilog Mode for Emacs, in addition to an embedded python processor for code generation. There are slight differences between the syntax for VHDL and Verilog, but the detailed documentation can be found for Verilog and VHDL here.

There are 2 ways to expand the templates

  • To expand the templates for the file only, right click in the editor window to open the menu shown below

  • To expand the templates for the whole project, right click in the project window to open the menu as shown below

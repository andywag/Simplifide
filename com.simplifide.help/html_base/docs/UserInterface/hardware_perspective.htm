The hardware perspective view is a view which contains the elements associated with viewing the important aspects of the design.

The view consists of 4 views, 2 of which are customized for hardware applications.
  1. The navigator view contains the views of the file and project hierarchy
  2. The chip hierarchy view contains a view of the hierarchy of the chip.
  3. The problems view contains a listing of all of the errors in the design
  4. The outline view contains a tree based listing of the items in the design

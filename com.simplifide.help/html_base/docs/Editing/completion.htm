Code completion allows the user to select from the objects which are in context. Completion can be activated in 3 ways.

  1. Typing Ctrl+Space automatically opens the completion
  2. Typing a period "."
  3. Typing a left paren "("

There are 2 reserved words which have a meaning for completion

  • this : refers to the current context
  • root : refers to the project context

There is also a preference page for selecting some options for the templates and the colors associated with the pop-up screen. To activate the preference page select Window->Preference from the top menu and open the SimplifIDE/Editors/Content Assist Page

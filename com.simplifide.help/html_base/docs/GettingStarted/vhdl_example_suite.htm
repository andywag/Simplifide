There is an example Vhdl project which can be created by a wizard. This is a good way to get started examining how the basic platform works.

  1. Select File->New->Project from the menu
  2. Choose Hardware/Verilog or Hardware/VHDL Example Project from the dialog box

  3. Select any name. Suite names unlilke Project names are arbitrary
  4. If the Hardware Perspective is not open choose yes from the Hardware Perspective Menu
  5. Look at the Features of the Hardware Perspective

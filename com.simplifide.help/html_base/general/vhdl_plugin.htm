The eclipse plugin for Hardware Design brings the ease and utility of a cutting edge java IDE to VHDL and Verilog. The main features of the plugin are shown below with many additional features in progress.

  • Syntax Coloring
  • Error Highlighting / Linting
  • Code Folding
  • Code Completion
  • Template Generation
  • Design Outline View
  • Hierarchical Project View
  • Hyperlinks
  • Text Hovers / HdlDoc
  • Context Based Searching
  • Refactoring / Project Wide Rename
  • Simulator Interface
